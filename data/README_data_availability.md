# Data availability

The primary breast tissue microarray dataset used by this workflow is publicly available from Zenodo.

DOI: 10.5281/zenodo.808456

Raw and preprocessed imaging data are not distributed in this repository because of file size and provenance constraints. The CSV files in `data/templates/` document the expected metadata structure for running or adapting the workflow.

Before running the MATLAB scripts, download the relevant data locally and update the paths in:

```text
matlab/arbiter_config_paths.m
```
